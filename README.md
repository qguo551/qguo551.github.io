# Qiwei Guo Research Homepage

This repository hosts the GitHub Pages site for Qiwei Guo:

https://qguo551.github.io

The site is intentionally lightweight and static. It presents research interests,
selected public projects, and contact links without requiring a build step.

## Edit Guide

- Update personal role, affiliation, email, and publication details in
  `index.html` when verified.
- Keep project descriptions tied to public repositories unless stronger
  source material is available.
- Avoid adding unverified paper titles, positions, or institutional claims.

## Local Preview

Open `index.html` directly in a browser, or run a small static server from this
folder:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.
