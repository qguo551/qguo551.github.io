# Qiwei Guo Research Blog

This repository hosts the GitHub Pages research blog for Qiwei Guo:

https://qguo551.github.io

The site is intentionally lightweight and static. It presents Qiwei Guo's
research positioning, blog notes, selected public projects, and contact links
without requiring a build step.

## Edit Guide

- Add new blog entries under `posts/` and link them from the Blog section in
  `index.html`.
- Update publication details only after title, author list, year, and source
  have been verified.
- Keep project descriptions tied to public repositories unless stronger
  source material is available.
- Avoid adding unverified paper titles, positions, or institutional claims.

## Local Preview

Open `index.html` directly in a browser, or run a small static server from this
folder:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.
